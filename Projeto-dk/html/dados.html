Os Dados — Modelo de Banco

As tabelas essenciais incluem Usuário, Título, Histórico e Recomendação.

O Diagrama Entidade-Relacionamento (DER) representa como os dados principais do sistema de recomendação são organizados e conectados. Embora a Netflix real utilize um sistema muito mais complexo, este modelo foi simplificado para atender somente às necessidades do trabalho, focando na estrutura mínima necessária para gerar recomendações personalizadas.

O DER demonstra como as informações são distribuídas e como cada entidade se relaciona com as demais. No modelo criado, trabalhamos com cinco entidades principais:

2. Conteúdo:

Representa todos os filmes, séries e produções disponíveis no catálogo da plataforma. Cada conteúdo pertence a um gênero específico, mas pode aparecer no histórico de diversos usuários.

3. Gênero:

Responsável por classificar cada conteúdo, como ação, drama, comédia, romance, entre outros. Um único gênero pode estar associado a vários conteúdos diferentes.

4. Histórico de Visualização:

Registra tudo o que cada usuário assiste. Essa entidade faz a ligação entre Usuário e Conteúdo, armazenando informações importantes para o algoritmo, como data, horário e tempo assistido.

5. Recomendação:

Mostra quais títulos foram sugeridos aos usuários com base no perfil e comportamento de cada um. Cada recomendação conecta um usuário a um conteúdo que o sistema considera relevante para ele.
